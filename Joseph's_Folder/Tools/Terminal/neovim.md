# NeoVim   
(NvChad) configuration and other stuff I need to note   
{leader} = space   
{s} = shift   
Essential stuff :    
  Open FileTree : ctrl + {N}   
  Themes : Spc + {T} + {H}   
  Save File : {:w} \*name if unititles   
  } FileTree commands :   

  New File : {A}   
  Copy File : {C}   
  Paste File : {P}   
  Rename : {R}   
  Find Files : SPC + {F} + {F}   
  Find Opened buffer files : SPC + {F} + {B}   
  Terminal : SPC + {V}   
  Hide Line-numbers : SPC + {N}   
     
    
} Vim motions :    

X : delete character    
Dd : delete line   
Dw : delete word
